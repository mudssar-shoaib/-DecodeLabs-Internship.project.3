# Interactive Hub — DecodeLabs Project 3

An impressive, all-in-one interactive webpage built with pure **HTML, CSS, and JavaScript** — no frameworks, no build tools.

## ✨ Features (All using DOM Manipulation)
1. 🌙 **Dark / Light Mode Toggle** — with localStorage persistence
2. 🔢 **Live Counter** — increment, decrement, reset with color feedback
3. ✅ **To-Do List** — add / complete / delete tasks, saved in browser
4. 🖼️ **Dynamic Image Gallery** — filter by category + lightbox preview
5. 💬 **Random Quote Generator** — click to inspire yourself
6. 🧮 **Mini Calculator** — full arithmetic keypad
7. 📱 **Responsive design** — works on mobile & desktop
8. 🎨 **Smooth animations & hover effects**

## 🚀 How to Run
Just **double-click `index.html`** — it opens in your browser. No installation needed.

Or right-click → *Open with* → your browser (Chrome / Edge / Firefox).

## 📂 Files
- `index.html` — page structure
- `style.css` — styling & theming
- `script.js` — all interactive logic

## 🎓 Concepts Demonstrated
- Event listeners (`click`, `keydown`)
- DOM manipulation (`createElement`, `innerHTML`, `classList`)
- LocalStorage for persistence
- Array methods (`filter`, `map`, `forEach`)
- CSS variables for theming
- Responsive layout with CSS Grid & Flexbox
